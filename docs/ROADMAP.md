# Roadmap

## Pre-Phase 1

Dataset access, safe downloaders, manifests, query templates, synthetic tests.

## Phase 1

Text-to-image retrieval index over RxRx19a and RxRx1, perturbation-level aggregation, metrics, random/shuffled baselines.

## Phase 2

Real local image support, rendered microscopy composites, and zero-shot VLM baselines using OpenCLIP/BiomedCLIP. See `PHASE2_REAL_DATA_AND_MODELS.md`.

## Phase 3

Lightweight alignment from biomedical text embeddings to microscopy/profile embeddings.

## Phase 4

JUMP-CP expansion.

## Phase 5

PERISCOPE optical pooled screening bridge.

## Phase 6

Prototype app and publication-quality benchmark artifacts.
